What are HTML entities? Give some examples. HTML entities are special codes used to represent characters that have special meaning in HTML. They are typically used to display characters that are reserved or have special formatting purposes in HTML. Here are some examples: < represents the less-than symbol (<). > represents the greater-than symbol (>). & represents the ampersand symbol (&). " represents the double quotation mark ("). © represents the copyright symbol (©). € represents the euro symbol (€). ® represents the registered trademark symbol ® ™ represents the trademark symbol ™ ' represents ' (apostrophe or single quotation mark)   represents a non-breaking space
